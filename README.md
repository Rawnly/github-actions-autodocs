# Github Actions Autodocs (GCA)

## Installation

Install via cargo

```sh
  cargo install github-actions-autodocs
```

## Usage

```sh
Usage: github-actions-autodocs --file <FILE>

Options:
  -f, --file <FILE>  Source path
  -h, --help         Print help
  -V, --version      Print version
```
